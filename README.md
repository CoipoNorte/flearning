# 🟢 flearning

Guía interactiva **hecha con Vue 3** para aprender frameworks y herramientas frontend modernas: Vue 3, Svelte, TypeScript y Sass. Sidebar colapsable, código copiable y estilo Vue.

![Vue](https://img.shields.io/badge/Vue_3-42b883?logo=vuedotjs&logoColor=white)
![Svelte](https://img.shields.io/badge/Svelte-FF3E00?logo=svelte&logoColor=white)
![TypeScript](https://img.shields.io/badge/TypeScript-3178C6?logo=typescript&logoColor=white)
![Sass](https://img.shields.io/badge/Sass-CC6699?logo=sass&logoColor=white)
![Tailwind](https://img.shields.io/badge/Tailwind_CSS-v4-06b6d4?logo=tailwindcss)

## 🔗 Temas cubiertos

**Vue 3:** Intro · Setup · Templates · Reactividad · Componentes · Props · Lifecycle · Computed · Router · Pinia

**Svelte:** Intro · Sintaxis · Reactividad · Componentes · Stores · Comparación

**TypeScript:** Intro · Tipos · Interfaces · Funciones · Generics · TS+React/Vue

**Sass:** Intro · Variables y Nesting · Mixins · Módulos

## 🛠️ Tecnologías

- **Vue 3** — Composition API (script setup)
- **Tailwind CSS v4** — Estilos utility-first
- **Vite** — Build tool
- **Lucide Vue Next** — Iconos

## 🚀 Ejecutar

```bash
npm install
npm run dev
```
